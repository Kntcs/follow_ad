## 7. [2501.15830] Spatialvla: Exploring spatial representations for visual-language-action model

**一句话总结**: 提出空间感知的VLA模型，通过3D位置编码和自适应动作网格实现跨机器人的空间理解和泛化控制

---

### 背景与动机

当前VLA（Vision-Language-Action，视觉-语言-动作）模型主要依赖2D图像输入，缺乏对3D物理世界的精确感知和理解。人类在操作物体时会本能地构建丰富的3D空间心智表征，但现有机器人基础模型难以实现这种空间智能。具体存在两大挑战：

1. **观测不对齐**：不同机器人的相机传感器安装位置各异（腕部相机 vs 第三人称相机），导致观测空间无法跨机器人统一
2. **动作异构性**：不同机器人的自由度、运动控制器、工作空间配置差异巨大，难以学习可迁移的空间动作知识

这些异构性问题使得现有通用策略在3D空间理解能力上严重滞后。

---

### 核心创新

**SpatialVLA通过两个关键设计将3D空间智能注入VLA模型：**

1. **Ego3D Position Encoding（自我中心3D位置编码）**
   - **做了什么**：为2D视觉特征注入3D空间上下文
   - **怎么做的**：使用ZoeDepth深度估计器从RGB图像获取深度图，通过相机内参反投影计算每个像素在自我中心坐标系中的3D位置，将3D位置通过正弦编码+MLP融合到SigLIP视觉特征中
   - **为什么这样设计**：自我中心坐标系无需相机-机器人外参标定，天然适配所有机器人embodiment

2. **Adaptive Action Grids（自适应动作网格）**
   - **做了什么**：用自适应空间网格离散化连续动作空间
   - **怎么做的**：对整个预训练数据集的动作分布拟合高斯分布，将每个动作维度按等概率划分成M个区间（translation用球坐标ϕ/θ/r，rotation用欧拉角roll/pitch/yaw），每个网格学习可微的空间动作token
   - **为什么这样设计**：统计驱动的网格划分捕获真实机器人运动特性，使跨机器人动作知识可迁移；相比RT-2/OpenVLA的7个token，只需3个token（translation、rotation、gripper各1个），推理速度提升2-3倍

3. **Spatial Embedding Adaption（空间嵌入自适应）**
   - **做了什么**：预训练模型快速适配新机器人的创新方法
   - **怎么做的**：在新机器人数据上重新拟合高斯分布并重新划分网格，用三线性插值从预训练网格嵌入初始化新网格
   - **为什么重要**：无需从头训练，通过网格重离散化实现参数高效的迁移学习

### 技术方案

#### 整体架构

- **骨干网络**：基于PaliGemma 2（3.5B参数）视觉-语言模型
- **输入**：单个第三人称RGB图像 + 自然语言任务指令
- **输出**：自回归生成3个空间动作token（对应translation/rotation/gripper）
- **训练目标**：标准交叉熵next-token prediction（公式1）

#### 关键技术点

**1. Ego3D Position Encoding详解**

```
输入：RGB图像 I
步骤1：深度估计 D = ZoeDepth(I)
步骤2：像素(u,v)反投影到3D空间
       x = (u - cx) * D(u,v) / fx
       y = (v - cy) * D(u,v) / fy  
       z = D(u,v)
步骤3：SigLIP提取2D特征 X ∈ R^(d×h×w)
步骤4：3D位置编码
       P' = MLP(γ(P))  其中γ是正弦位置编码
步骤5：特征融合
       O_3d = X + P'   (公式2)
```

**为什么用正弦编码**：保持空间连续性，相邻位置的编码向量在嵌入空间中距离近

**2. Adaptive Action Grids详解**

**动作空间定义**：
- 7D连续动作 a = {x, y, z, roll, pitch, yaw, grip}
- 分解为3部分：translation Δ_T、rotation Δ_R、gripper G

**离散化策略**：
```
步骤1：归一化到[-1,1]
步骤2：统计全数据集动作分布，拟合高斯 N(μ_a, Σ_a)
步骤3：等概率网格划分（公式4）
       找到切分点 a_1,...,a_M 使每个区间概率=1/M
       
例如：translation用球坐标(ϕ,θ,r)
     M_ϕ × M_θ × M_r 个3D网格
     rotation用欧拉角(roll,pitch,yaw)  
     M_roll × M_pitch × M_yaw 个3D网格
```

**token数量优化**：
- RT-2/OpenVLA：7个维度×256 bins = 7个token/action
- SpatialVLA：1个translation token + 1个rotation token + 1个gripper token = 3个token/action
- **推理速度**：20Hz (SpatialVLA) vs 5.2Hz (OpenVLA) = **3.8倍加速**

**3. 训练框架**

**预训练阶段**：
- 数据：1.1M真实机器人episode（OXE子集 + RH20T）
- 硬件：64×A100 GPU，10天
- Batch size：2048
- Action chunk：T=4（预测4步未来动作）
- 冻结策略：文本token嵌入冻结（保持VLM的世界知识）
- 数据混合：前2/3包含DROID数据集，最后1/3移除DROID提升质量

**后训练阶段**：
- LoRA微调（r=32, α=32）
- **核心创新**：Spatial Embedding Adaption
  - 在新机器人数据上重新拟合N(μ_new, Σ_new)
  - 重新划分网格G_new
  - 三线性插值初始化新嵌入：e^new_i = Σ_j w_j * e_j（公式6）
  - 权重w_j由网格中心距离计算

**为什么这样初始化有效**：预训练的空间动作知识（如"向左移动10cm"）被编码在嵌入中，插值保证新网格继承临近的运动语义

#### 与现有方法的区别

| 维度 | RT-2/OpenVLA | Octo | SpatialVLA |
|------|--------------|------|------------|
| 观测表示 | 纯2D图像 | 2D图像 | 2D图像+3D位置编码 |
| 动作离散化 | 固定256 bins/维度 | 连续动作 | 自适应高斯网格 |
| Token数/action | 7个 | N/A | 3个 |
| 跨机器人迁移 | 全参数微调 | Transformer适配器 | 网格重离散化 |
| 推理速度 | 5.2Hz | 4.2Hz | 20Hz |

### 实验效果

#### 主要结果

**1. SimplerEnv仿真基准（零样本评估）**

**Google Robot任务（表I）**：
- Visual Matching：71.9%（vs RT-2-X 60.7%，OpenVLA 27.7%）
- Variant Aggregation：68.8%（vs RT-2-X 64.3%）
- **关键发现**：3.5B参数的SpatialVLA超越55B参数的RT-2-X

**WidowX任务（表II）**：
- 整体成功率：34.4%零样本，42.7%微调后
- "Put Eggplant in Basket"任务：**100%成功率**（微调后）
- 超越RoboVLM（31.3%微调）+11.4%

**2. 真实WidowX机器人（7项任务，图5）**：
- 平均成功率：领先所有基线
- **语言理解**：正确执行"把绿色杯子放在白盘子上（不是粉色布）"
- **运动鲁棒性**：人类干扰下仍能追踪和抓取胡萝卜/茄子
- **语义泛化**：未见过的背景/姿态下保持高成功率

**3. 新机器人适配**

**LIBERO仿真（表III）**：
- 平均成功率：78.1%（vs OpenVLA 76.5%，Octo 75.1%）
- LIBERO-Spatial（空间布局变化）：**88.2%**（最高）
- LIBERO-Long（长时域任务）：55.5%（仍有提升空间）

**Franka真机（13任务，图6）**：
- 单任务：82%（vs Diffusion Policy 81%，OpenVLA 74%）
- 指令遵循：70%（vs OpenVLA 58%，**+12%**）
- 多任务泛化：57%（OpenVLA/Octo < 40%）

**4. 空间理解能力专项测试（图7）**

**零样本WidowX**：
- "把最靠近机器人的玩偶放在车上"：理解空间关系词
- "杯子高度变化"：3D感知适应物体高度差异

**微调Franka**：
- "锅高度变化"场景：胡萝卜放入不同高度盘子，保持高成功率

**结论**：Ego3D编码显著增强复杂空间布局理解

#### 消融实验（重要发现）

**表IV：SimplerEnv Google Robot消融**

| 变体 | Visual Matching | Variant Aggregation |
|------|-----------------|---------------------|
| 完整SpatialVLA | **71.9%** | **68.8%** |
| 无Ego3D编码 | 60.0% (-11.9%) | 57.0% (-11.8%) |
| 无自适应网格（固定256 bins） | 63.3% (-8.6%) | 60.2% (-8.6%) |

**关键洞察**：
1. **Ego3D编码贡献最大**（~12%提升），验证3D空间理解的核心价值
2. **自适应网格 vs 固定bins**：高斯驱动的网格划分提升8.6%
3. **3 token vs 7 token**：token压缩在保持性能的同时提速3.8倍

**表V：迁移学习对比**

| 微调方法 | LIBERO-Spatial | Franka多任务 |
|----------|----------------|--------------|
| Spatial Embedding Adaption | **88.2%** | **57%** |
| LoRA全参数微调 | 84.7% | 52% |
| 冻结骨干 | 78.9% | 43% |

**结论**：网格重离散化 + 三线性插值初始化是最优后训练策略

### 个人点评

**亮点**：
1. **系统性强**：首次在VLA中同时解决观测和动作的3D对齐问题，形成完整闭环
2. **工程优雅**：3 token设计兼顾性能和效率，20Hz推理速度满足实时控制
3. **泛化出色**：1.1M数据预训练后，零样本胜过多数微调后的基线

**局限**：
1. 依赖ZoeDepth深度估计，反光/透明物体可能失效
2. 长时域任务（LIBERO-Long 55.5%）仍落后，需引入时序建模
3. 自适应网格依赖数据分布统计，数据偏少的新机器人可能次优

**未来方向**：
- 结合立体视觉或主动深度传感器获取更准确3D信息
- 探索Transformer时序层处理长视野任务
- 研究多模态3D表示（点云、voxel、神经辐射场）
