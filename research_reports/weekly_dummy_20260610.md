个去噪时间步，采样 25% 的查询 token 计算近似注意力图，通过公式 (3) 计算帧注意力分数矩阵 F ∈ R^{num_head × 3}。
3. **DHP 分类**：执行贪心算法，计算 ℓ_h = max(F_{h,0}, F_{h,1})，排序后选择最小的 N 个为 dummy heads，剩余头根据 F_{h,0} 和 F_{h,1} 分为 sink/neighbor。
4. **HMA 缓存管理**：在后续 AR 步中，根据头分类结果为每类头维护不同的 KV cache：
   - Sink heads：保留 sink frame + current frame
   - Neighbor heads：保留 L-1 个邻近帧 + current frame
   - Dummy heads（PAF 版本）：保留 t-1 frame + current frame
5. **注意力计算**：将头按类型索引分组，分别执行对应的注意力计算（公式 4 和 7），输出重新拼接回原位置。使用 Triton 优化索引和拼接操作，减少额外开销。

**3.4 与现有方法的区别**

| 方法 | 压缩粒度 | 自适应性 | 额外开销 | 压缩比 | 视频特化 |
|------|---------|---------|---------|--------|---------|
| R-KV / Infinipot-V | Token 级 | 每步重新计算 | 高 | 16.7% | 否 |
| TeaCache | 时间步跳过 | 静态 | 低 | N/A | 是 |
| Dummy Forcing | Head 级 + Frame 级 | 一次分类多步复用 | 极低（100ms）| 27.8% | 是 |

核心差异：发现头级别异质性；帧级聚合避免 token 遍历；训练无关+自适应；更激进压缩但更高加速。

## 9. [2509.19200 v1] Dummy Atom Based Thermodynamic Integration Accurately Predicts Disulfide Redox Free Energies in Proteins

### 1. Motivation（研究动机）

二硫键形成/断裂是蛋白质氧化还原调控核心，准确预测其氧化还原电位对理解功能至关重要。实验方法成本高，计算方法中CGI方法在小蛋白上R²仅0.02，且无法处理大蛋白。问题根源：力场无法描述共价键断裂，现有方法未充分利用热力学积分精度。

### 2. Contribution（核心贡献）

1. **Dummy atom + TI框架**：首次结合dummy氢原子与热力学积分，小蛋白R²从0.02提升至0.996
2. **大蛋白验证**：成功预测147.7 kDa GAPDH（1352残基），偏差仅2.5 kJ/mol
3. **PTM-Psi工具**：自动化流程集成，降低使用门槛
4. **机制揭示**：局部相互作用误差在大蛋白中被稀释，方法对大体系更准确

### 3. Method（技术方法）

**核心技术**：
- **Dummy atom（CYZ残基）**：插入零电荷虚拟H，使S-S键通过参数变化转为S-H（避免拓扑不连续）
- **Dual Topology TI**：30步λ窗口分两阶段（静电→vdW+键合），BAR积分∫⟨∂U/∂λ⟩dλ
- **1-2/1-3/1-4相互作用转换**：state A中被S-S键屏蔽的非键相互作用在state B中"打开"
- **NeRF自动插入**：碰撞检测优化dummy位置，~O(N_res²)复杂度

**算法流程**：体系准备→100ns NPT平衡→30步TI采样（1ns/步）→BAR积分→能斯特方程转换电位

**与CGI区别**：TI基于平衡态直接积分自由能梯度，CGI依赖非平衡拉伸+Crooks定理外推，前者统计更稳健。

### 4. Experiment（实验验证）

**数据集**：4个小蛋白（82-118残基）+ 1个大蛋白GAPDH（1352残基）

**结果**：

| PDB | 实验 ΔG | TI计算 ΔG | CGI计算 ΔG | TI偏差 | CGI偏差 |
|-----|---------|----------|-----------|--------|--------|
| 1FOV | 38.21 | 12.01 | - | 26.20 | - |
| 7Q53（GAPDH） | 58.27 | 55.79 | n/a | 2.48 | - |

关键发现：
- 小蛋白绝对偏差15-26 kJ/mol（但R²=0.9996），大蛋白仅2.5 kJ/mol
- S-S距离收敛至3.6 Å（匹配还原态晶体3.67 Å）
- 局部1-4 Q相互作用未完全转换是小蛋白误差主因

**局限性**：GROMACS不支持静电短程→长程转换；小蛋白需线性校正；单二硫键限制。
